File LTexture:
void LTexture::Render(int x, int y, SDL_Renderer* gRenderer, SDL_Rect* clip)
{
	SDL_Rect renderSpace = { x, y, mWidth, mHeight }; //x,y ở đây là bị trí renderSpace xuất hiện so với window

	if (clip != nullptr)// vị trị frame đã được khởi tạo ở hàm Load ở file main
	{
		renderSpace.w = clip->w;
		renderSpace.h = clip->h;
	}
	
	SDL_RenderCopy(gRenderer, mTexture, clip, &renderSpace);// copy 1 phần từ texture xong gán nó qua renderSpace
    // Khi này cái vòng for ở Load mới có tác dụng
}

2.\
offsetSpeed is used to create a loop in the game by continuously 
changing the position of the background texture on the screen, creating the illusion of movement. 
The idea is to move the texture to the left every frame by a certain speed value, 
and when the texture has fully scrolled off the screen to the left, 
we reset the offset speed to 0 and render the texture again at its starting position to create an endless scrolling effect.

offsetSpeed is decreased by a constant speed value layer_speed each frame, causing the background texture to move to the left.
